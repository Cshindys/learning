Storage key: learning_app_data_v1
Instructions:
  1. Open the page served from the same origin (e.g., http://localhost:3000) so localStorage is consistent.
  2. After adding students/questions, refresh the browser — data should persist across refresh and PC restarts.
  3. Use "Export (CSV)" to download questions. For Excel (.xlsx), add SheetJS (see below) then press Export (.xlsx).
Add SheetJS for real .xlsx export:
 <script src="https://cdn.jsdelivr.net/npm/xlsx/dist/xlsx.full.min.js"></script>
Add the script tag above before the persistence script to enable export to .xlsx.

Quick developer tips

  • If you find data disappears after a full OS restart, confirm you are loading the site from the same origin and not in a private window that clears storage.
  • Check DevTools → Application → Local Storage to inspect the saved data.
